# Obtaining an API Token for Llama API
## Step 1: Access our Website
To access our website, you must first Create an Account. To do this,
visit [https://www.llama-api.com](https://www.llama-api.com), click on
*Log In* --> *Sign up* and follow the steps on the screen.

<img height="200" src="/images/signup-screen.png" alt="signup" />

## Step 2: Waitlist
Llama is currently in a Private Beta; so, when you signup, you are added to our waitlist.
Once you receive our invitation, you can follow the next steps on this Doc to become one
of our first customers!

> :calendar: Usually we are able to approve your profile in a few days, so keep an eye out for our email!

### Step 3: Obtain an API Token

Once your registration is complete and your account has been approved, log in and navigate to [API Token](https://www.llama-api.com/account/api-token).

<img height="200" src="/images/login.png" alt="token button" />

On this page, you will find your API Token, as shown in the image below.

<img height="200" src="/images/encrypted-token.png" alt="encrypted" />

Hover over the clipboard icon and copy your token.

<img height="200" src="/images/token-api.png" alt="token" />

Now, you are ready to be one of the first testers of Llama API! Follow the [Quick Start Guide](/quickstart) for more information.
