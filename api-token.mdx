# How to Get the Llama API Token

## Accessing the Llama API

You must have an account on the Llama API.

### Sign Up

Visit the [signup](https://www.llama-api.com) page by clicking on Log In.

<img height="200" src="/images/signup-screen.png" alt="signup" />

After registration, you will be added to the waiting list.

### Token

Once your registration is complete and your account has been approved, log in and navigate to [API Token](https://www.llama-api.com/account/api-token).

<img height="200" src="/images/login.png" alt="token button" />

On this page, you will find your Token.

<img height="200" src="/images/encrypted-token.png" alt="encrypted" />

Hover over the clipboard icon to copy your token.

<img height="200" src="/images/token-api.png" alt="token" />

Now, you can start using the Llama API. Follow the [quick start guide](/quickstart) for more information.
